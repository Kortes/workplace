Настройка рабочего места на Ubuntu
==================================

Структура катаголов
-------------------

* `bin`     - Скрипты доступные мне из консоли
* `etc`     - Настройки для компонентов системы
* `doc`     - Различная документация
* `website` - Web-сайты
* `project` - Другие проекты: библиотеки, приложения
